# Hospital Management System (County Hospital - Mombasa)

This workspace contains a production-ready scaffold for a Hospital Management System (HMS) tailored for a Kenyan county hospital.

Quick start (dev):

1. Copy `.env.example` to backend/.env and update values.
2. Run with Docker Compose:

```bash
docker-compose up --build
```

Backend: `http://localhost:4000`
Frontend: `http://localhost`
